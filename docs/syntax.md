# Синтаксис языка

## Лексика

- **Комментарии**: `# …` или `-- …`
- **Числа**: `123`, `0`, `42`
- **Идентификаторы**: буквы, цифры, `_`, начинаются с буквы или `_`
- **Операторы**: `+ - * / == = \ |> [ ] ( ) ,`

## Основные конструкции

| Конструкция      | Пример                             |
| ---------------- | ---------------------------------- |
| Lambda           | `\\x -> x + 1`                    |
| Применение       | `f x y`                            |
| Let              | `let x = 10 in x * 2`              |
| Let rec          | `let rec fact n = … in fact 5`     |
| If-выражение     | `if cond then a else b`            |
| Списки           | `[1,2,3]`, диапазон `[0..10)`      |
| Pipeline         | `xs |> map (_*2) |> filter (>5)`   |
