# Установка

## Зависимости

- GCC (поддержка C17)
- make

## Сборка

```bash
git clone https://github.com/ваш-аккаунт/rephi.git
cd rephi
make
```
`rephi` появится в корне.

`rephi` появится в корне.

## Установка stdlib

```bash
make install-stdlib DESTDIR=/usr/local
```

Стандартная библиотека установится в
`/usr/local/share/rephi/stdlib`
